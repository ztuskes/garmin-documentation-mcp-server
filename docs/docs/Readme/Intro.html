Introduction

Connect IQ System None (API 8.2.3)

Released on August 11, 2025

Connect IQ is the first-ever open platform for third-party developers to create apps for Garmin products. With Connect IQ, you can build on the features that have made Garmin products an integral part of customersâ€™ active lifestyles:

  • Customize the base watch experience with colorful watch faces

  • Translate rich sensor data into new metrics and insights with custom data fields

  • Provide information at a glance with widgets

  • Offer solutions for active lifestyles with apps

  • Manage and playback audio content, like music, podcasts, and audio books, from third-party content providers.

Connect IQ apps are developed in Monkey C, an object-oriented language built from the ground up for easy Connect IQ app development on Garmin devices, allowing developers to focus more on the customer and less on resource constraints.

This documentation is broken into five sections:

  1. Connect IQ Basics - An overview of the Connect IQ system that provides the lay of the land.

  2. Monkey C - Learn the language you didn't know you already knew.

  3. API Documentation - Detailed information about the Monkey C APIs.

  4. Core Topics - A deep dive into the Connect IQ system and the APIs it offers.

  5. Connect IQ FAQ - Pro-tips and frequently asked questions about Connect IQ.

  6. Reference Guides - Deep dives into Connect IQ tools and topics.

  7. Device Reference - Details about supported devices.
